伸縮変形をさせるAviutl2のスクリプトです。  
「変形」の所に表示させるようになっています。  
縦、横、などを伸縮させて変形させるスクリプトです。  
上下のみ、上のみ、下のみ、左右のみ、左のみ、右のみ、だけを変形させて伸ばすことができます。
